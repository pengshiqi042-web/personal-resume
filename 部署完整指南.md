## 🌐 第三步：创建 GitHub 仓库

1. **访问 GitHub**：https://github.com
2. **登录账号**（如果没有，先注册一个免费账号）
3. **创建新仓库**：
   - 点击右上角 "+" 号
   - 选择 "New repository"
4. **填写仓库信息**：
   - **Repository name**: `personal-resume`（或你喜欢的名称）
   - **Description**: 个人简历网站（可选）
   - **选择 Public**（公开，免费）
   - ⚠️ **不要勾选** "Initialize this repository with a README"
   - ⚠️ **不要勾选** "Add .gitignore"
   - ⚠️ **不要勾选** "Choose a license"
5. **点击 "Create repository"**

---

## 🚀 第四步：推送代码到 GitHub

### 方法一：使用脚本（推荐）

1. **双击运行 `Git推送助手.bat`**
2. **按提示输入**：
   - GitHub 用户名
   - 仓库名称（例如：personal-resume）
3. **等待推送完成**

### 方法二：手动运行命令

在项目根目录运行（替换为你的信息）：

```bash
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

**示例**：
```bash
git remote add origin https://github.com/zhangsan/personal-resume.git
git branch -M main
git push -u origin main
```

---

## ✨ 第五步：在 Vercel 部署（获得公开网址）

### 详细步骤：

1. **访问 Vercel**：https://vercel.com

2. **注册/登录**：
   - 点击 "Sign Up" 或 "Log In"
   - 选择 **"Continue with GitHub"**（使用 GitHub 账号登录）
   - 授权 Vercel 访问你的 GitHub 账号

3. **导入项目**：
   - 登录后，点击 **"Add New..."** → **"Project"**
   - 在 "Import Git Repository" 中，找到你刚创建的仓库
   - 点击 **"Import"**

4. **配置项目**（通常会自动识别，检查一下即可）：
   - **Framework Preset**: `Vite`（应该自动识别）
   - **Root Directory**: `./`（默认）
   - **Build Command**: `npm run build`（应该自动识别）
   - **Output Directory**: `dist`（应该自动识别）
   - **Install Command**: `npm install`（应该自动识别）

5. **部署**：
   - 点击 **"Deploy"** 按钮
   - 等待 1-2 分钟
   - 看到 ✅ 部署成功！

6. **获得网址**：
   - 部署完成后，你会看到一个网址，例如：
     - `https://personal-resume.vercel.app`
   - 或者 `https://personal-resume-xxx.vercel.app`
   - **这个网址就是你的个人简历网站！**

## ❓ 常见问题

### Q: 推送代码时提示需要登录？
A: 需要配置 Git 凭据：
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### Q: 推送失败，提示权限错误？
A: 检查：
1. GitHub 仓库名称是否正确
2. 仓库是否已创建
3. 是否有推送权限

### Q: Vercel 部署失败？
A: 检查：
1. GitHub 仓库是否已正确推送
2. 构建命令是否正确（`npm run build`）
3. 输出目录是否正确（`dist`）

### Q: 部署后图片不显示？
A: 确保所有图片都在 `public` 文件夹中，路径以 `/` 开头
