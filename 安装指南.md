# 个人作品集网站 - 安装指南

## 第一步：检查 Node.js 是否已安装

打开命令提示符（CMD）或 PowerShell，运行：

```bash
node --version
npm --version
```

如果显示版本号，说明已安装。如果没有，请先安装 Node.js：
- 访问 https://nodejs.org/
- 下载并安装 LTS 版本

## 第二步：安装项目依赖

在项目目录（`C:\Users\11727\Desktop\作品集`）中打开终端，运行：

```bash
npm install
```

这会安装所有必需的依赖包（React、Vite 等），可能需要几分钟时间。

## 第三步：启动开发服务器

安装完成后，运行：

```bash
npm run dev
```

你会看到类似这样的输出：
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## 第四步：在浏览器中查看

打开浏览器，访问：**http://localhost:5173/**

你应该能看到你的个人作品集网站了！

## 常用命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run preview` - 预览生产版本

## 遇到问题？

1. **npm install 失败**
   - 检查网络连接
   - 尝试使用国内镜像：`npm install --registry=https://registry.npmmirror.com`

2. **端口被占用**
   - Vite 会自动尝试其他端口
   - 或手动指定端口：`npm run dev -- --port 3000`

3. **依赖安装慢**
   - 使用淘宝镜像：`npm config set registry https://registry.npmmirror.com`
