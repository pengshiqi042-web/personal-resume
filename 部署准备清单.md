## 📋 部署步骤（使用 Vercel - 推荐）

### 第一步：检查项目是否可以构建

在项目根目录运行：
```bash
npm run build
```

如果构建成功，会生成 `dist` 文件夹。

### 第二步：初始化 Git 仓库（如果还没有）

```bash
git init
git add .
git commit -m "Initial commit: 个人简历网站"
```

### 第三步：创建 GitHub 仓库

1. 访问 https://github.com
2. 登录你的账号
3. 点击右上角 "+" → "New repository"
4. 仓库名称建议：`personal-resume` 或 `resume-website`
5. 选择 **Public**（公开，免费）
6. **不要**勾选 "Initialize this repository with a README"
7. 点击 "Create repository"

### 第四步：推送代码到 GitHub

在项目根目录运行（替换为你的GitHub用户名和仓库名）：

```bash
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

### 第五步：在 Vercel 部署

1. 访问 https://vercel.com
2. 点击 "Sign Up" 或 "Log In"
3. 选择 "Continue with GitHub"（使用GitHub账号登录）
4. 授权 Vercel 访问你的 GitHub 仓库
5. 登录后，点击 "Add New..." → "Project"
6. 在 "Import Git Repository" 中选择你刚创建的仓库
7. 配置项目（通常会自动识别）：
   - **Framework Preset**: Vite（会自动识别）
   - **Build Command**: `npm run build`（会自动识别）
   - **Output Directory**: `dist`（会自动识别）
   - **Install Command**: `npm install`（会自动识别）
8. 点击 "Deploy" 按钮
9. 等待 1-2 分钟，部署完成后会显示：
   - ✅ 部署成功
   - 🌐 你的网站网址（例如：`https://your-project.vercel.app`）

### 第六步：分享你的网站

部署完成后，你会得到一个公开的网址，例如：
- `https://your-project.vercel.app`

这个网址可以直接分享给任何人访问！

## 🔄 更新网站内容

以后如果需要更新网站内容：

1. 修改代码
2. 提交更改：
   ```bash
   git add .
   git commit -m "更新内容描述"
   git push
   ```
3. Vercel 会自动检测到更改并重新部署（通常1-2分钟）

## 📝 注意事项

1. **确保所有图片都在 `public` 文件夹中**
2. **确保 `resume.pdf` 在 `public` 文件夹中**（用于下载简历功能）
3. **首次部署可能需要几分钟时间**
4. **部署后可以随时更新内容**
